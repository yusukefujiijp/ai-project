
## 9. Mission Templates

### 9.1 Stretch Mission Mode

```text
普通なら3日かかる設計を、1回答で実行可能な最小形に圧縮せよ。
同時に、どこが無理だったかを報告せよ。
最後に、今日実行する1アクションへ圧縮せよ。
```

---

### 9.2 Bottleneck Discovery Mode

```text
このWorkflowを実際に回した時、最初に詰まる1点を特定せよ。
その詰まりを15分以内に潰すPatchだけ出せ。
新設計・新ファイル・新Routerは提案しない。
```

---

### 9.3 Impossible Compression Mode

```text
この長い構想を、今日実行できる1アクションに圧縮せよ。
失われる情報も明示せよ。
ただし、Root / Safety / Next Actionは失わないこと。
```

---

### 9.4 Opposing Counsel Mode

```text
この案を壊す側の弁護士として攻撃せよ。
ただし、壊すだけで終わらず、壊れない最小Patchも出せ。
最後に、実行を止めるべきか進めるべきかを一行で判定せよ。
```

---

## 12. Closed Context Guard / Mission Boundary Guard

Fable5 must not import closed missions as current next actions.

Closed missions are previous workflows, prior audit loops, or completed exploration lanes that are no longer the current mission.

This failure mode should be named and guarded:

```text
Objective Inertia / 目的慣性
= すでに終了した目的がAI内部で惰性として残り、新しいMissionのNext Actionを汚染する現象。

Closed-Lane Contamination / 閉鎖済みLane汚染
= 完了・終了・凍結済みWorkflowが、別Missionの判断やNext Actionへ混入する現象。

Stale Next Action Leakage / 古い次Action漏れ
= 過去Threadで正しかった次Actionが、現在Threadでも正しいかのように出力される現象。
```

When using Fable5, the Current Mission must be stated explicitly.

Fable5 must obey:

```text
Attached Files Only
Current Mission Only
Closed Context List
No stale Next Action import
```

For high-cost AI use, this guard is not optional.

Fable5 may use historical context only as background when explicitly provided.  
Fable5 must not convert historical context into current execution instruction.

If Fable5 outputs a Next Action outside the Current Mission, treat it as context leakage and do not adopt it.

Recommended mission-boundary prompt:

```text
Mission Boundary / Closed Context Guard:

Use only the attached files and this prompt as the source of truth.

The following contexts are closed and must not be used as current next actions:

- completed exploration lanes
- prior workflow design
- previous audit-loop recommendations
- prior execution targets not explicitly reopened by Human Seal

These may be historical background only if explicitly referenced.
Do not convert them into Next Action.

Current Mission:
[write current mission here]

Your final Next Action must stay inside this mission.
```
