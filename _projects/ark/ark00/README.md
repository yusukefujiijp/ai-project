# Ark00

## 0. Ark00とは

Ark00は、Ark Projectにおける **Selective Incubation / Project Embryo** である。

つまり、Ark99で観測されたSeedのうち、繰り返し現れ、軽く形を持ちはじめたものを、正式Project化の前に選抜・温存・軽量検証するための入口である。

```text
Ark00:
  すべてのSeedを受ける場所ではない。
  すぐ正式Project化する場所でもない。
  繰り返し現れたSeedを、軽く温める場所である。
```

Ark00は最終Sealではない。  
Ark00は正式Project化ではない。  
Ark00は、Projectになる可能性を持つSeedを、まだ胚の状態で観測するFieldである。

## 2. Ark99との違い

Ark99とArk00は、役割が違う。

```text
Ark99:
  Wild Seed Field。
  まだ分類しきれないSeedを受け、観測する。

Ark00:
  Selective Incubation。
  繰り返し現れたSeedを選び、温め、正式Project化前に軽く検証する。
```

短く言えば、こうである。

```text
Ark99:
  受ける場所。

Ark00:
  選び始める場所。
```

Ark00は、Ark99の上位互換ではない。  
Ark00は、Ark99に現れたすべてのSeedを自動的に昇格させる場所ではない。

## 3. Selective Incubation / Project Embryo

Ark00の中心概念は、**Selective Incubation / Project Embryo** である。

```text
Selective Incubation:
  すべてを育てるのではなく、
  繰り返し現れ、軽く形を持ちはじめたSeedだけを温める。

Project Embryo:
  正式Projectではないが、
  Project化の可能性を持つ胚の状態。
```

Ark00は、Seedを急いでProject化しない。  
しかし、Seedを放置もしない。

```text
Ark00 holds the embryo.
Ark00 does not force the birth.
```

---

## 4. Incubation Gate

Ark00には、**Incubation Gate** がある。

Ark99では、Second-Thread Gateが重要である。  
同じSeed・Theme・Named Patternが第二Threadにも現れた場合、それはArk00 pickup候補になる。

Ark00では、その次を観測する。

```text
Second-Thread Gate:
  二度目にも現れるかを見る。

Incubation Gate:
  そのSeedが、軽く形を持ち、
  正式Project候補として温存すべきかを見る。
```

Incubation Gateは、早すぎるProject化と、Seedの消失を同時に防ぐためのGateである。
