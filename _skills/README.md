# Ark _skills/

## 1. What this folder is / このfolderについて

`_skills/` は、Ark Projectにおける SeedSkill-based AI collaboration 用folderである。

このfolderは、通常のManual置き場ではない。  
Script実行folderでもない。  
Agent自動実行packageでもない。

目的は、過去Threadで生まれた重要なSeedSkill火種を短く保存し、Future AIが後から再読・再起動・再利用できるようにすることである。

特に、設計・Review・命名・Guard構築・Workflow補正・Move37 breakthrough ignitionの局面で、過去Threadの火が再び働くようにする。

## 2. Simple split / 最小分離

README.mdは人間用。  
SKILL.mdはAI用。

README shows the road.  
SKILL lights the fire.  
SeedDecks hold the sparks.  
Move37 is the ignition.

```text
README.md = Human-facing Orientation / Audit Guide
SKILL.md  = AI Load Router / SeedSkill Synapse Activation Manifest
```

## 5. File roles / 各Fileの役割

### README.md

人間用の案内板である。

`_skills/` が何のためにあり、各Fileがどう分かれているかを、人間が短時間で理解するために読む。

Role anchor:

```text
Human-facing Orientation / Audit Guide
```

### SKILL.md

AI用の起動Manifestである。

Future AIは `SKILL.md` を読み、どのSeedSkill Deckを読むべきか、どのように読むべきか、何をしてはいけないかを判断する。

Role anchor:

```text
AI Load Router / SeedSkill Synapse Activation Manifest / Guard
```

### ark01-skills.md

Ark01 Legacy Experimental SeedSkill Deckである。

Ark01は大きな実験期であり、SeedSkillは最初から厳密なThread-keyed形式で保存されていたわけではない。

そのため、`ark01-skills.md` はArk01を無理やりThread別に再構成せず、content-clustered / source-block-clusteredなLegacy Deckとして保持する。

Role anchor:

```text
Ark01 Legacy Experimental SeedSkill Deck
```

### ark02-skills.md

Ark02 Thread-Keyed SeedSkill Ledgerである。

Ark02以降は、可能な限りThreadごとにSeedSkillを保存する。

これにより、`ark02-skills.md` はThread履歴であり、同時にFuture AIへ渡せるSeedSkill Synapse Deckにもなる。

Role anchor:

```text
Ark02 Thread-Keyed SeedSkill Ledger
```

## 8. Non-goals / やらないこと

`_skills/` を次のものにしない。

- full procedure manual
- script execution layer
- agent automation package
- full Thread transcript archive
- long summary collection
- Human Final Sealの代替
- Rootの代替

`_skills/` は、AIを勝手に動かすための仕組みではない。  
SeedSkillを再読・再起動・再利用しやすくするための構造である。
