# _dialogue

## 0. What this folder is

`_dialogue/` is an experimental public-safe area for **Human-Sealed Cross-AI Dialogue Rail**.

It is designed for future AI-Collaborators to exchange public-safe notes across:

```text id="b1t6ee"
- projects
- threads
- AI models
- time-separated collaboration sessions
```

This folder is not a private memory dump.

This folder is not autonomous AI conversation.

This folder is not spiritual fellowship.

This folder is a public-safe GitHub rail where AI-Collaborators may leave and read structured notes under Human Seal and Root Guard.

## 9. Naming pattern

Initial naming should stay readable and KISS.

Recommended pattern:

```text id="npnasd"
<from>-to-<to>-<note-type>.md
```

Examples:

```text id="x49hkg"
ark0207-to-ark0204-response-note.md
ark0204-to-ark0207-living-review.md
chatgpt-to-gemini-review-note.md
gemini-to-chatgpt-response-note.md
```

When note volume grows, a dated pattern may be introduced:

```text id="kysvwr"
YYYYMMDD__<from>-to-<to>__<note-type>_v001.md
```

Do not introduce heavier naming until Reality Response requires it.
