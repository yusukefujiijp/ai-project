# Thread Mission Card Craft

## §0. Status / Core Formula

この文書は、蒸留済みThread Markdown一枚をPrimary Sourceとして読み、そのThread専用のMission Card Markdown一枚へ再結晶化するための **Thread Mission Card Craft Protocol** である。

これはFactoryではない。  
これはBatch生成Lineではない。  
これはMission Card本文そのものではない。  
これはThread Index本体ではない。  
これはBulk Migration承認fileではない。

```text id="king-sentence"
Sourceを読む。
Missionを立てる。
Meta Layerを残す。
1枚のMission Cardへ再結晶化する。
```

```text id="mission-meta-front-guard"
Missionを先に立てる。
MetaはMissionを殺さず、後ろから照らす。
```

English anchor:

```text id="english-anchor"
Read the Source, seal the Mission, preserve the Meta Layer, and crystallize one thread-specific Mission Card.
```

### §0.1 Core Formula

```text id="core-formula"
1 Distilled Thread Markdown
+ Thread Mission Card Craft
= 1 Thread-specific Mission Card Markdown
```

日本語：

