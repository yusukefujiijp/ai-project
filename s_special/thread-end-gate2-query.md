# 【Thread-End Workflow Gate 2: Thread Harvest / Handoff Builder Query v002】

## 0. Query Purpose

これは Thread-End Two-Gate Workflow の **Gate 2: Thread Harvest** です。

Gate2の目的は、Threadを普通に要約することではありません。  
目的は、現在Threadの意味・熱・命名・Move37候補・未精製Seed・Next Compassを回収し、次Threadが再起動できる `handoff.md` 起動Keyと `Full Rail: next_thread` を生成することです。

```text id="purpose"
Gate2:
  Thread Harvest
  Handoff Builder
  next_thread first query generator
```
