## 9. Handoff Builder

Gate2では、Thread Harvestの最後に、次Threadで読むべき `handoff.md` の起動Keyを明確にする。

```yaml id="handoff-builder"
handoff_builder:
  must_extract:
    - "Priority Read Order"
    - "current state"
    - "completed GitHub commits"
    - "unresolved not-now items"
    - "next first task"
    - "Reality Response needed"
    - "Root / Fruit Guard"

  must_not:
    - "replace handoff.md with full Harvest"
    - "put handoff update instructions inside Full Rail: next_thread"
```
