# 【Thread-End Workflow Gate 2: Thread Harvest / Handoff Builder Query】

## 0. Query Purpose

これは Thread-End Two-Gate Workflow の **Gate 2: Thread Harvest** である。

Gate2の目的は、Threadを普通に要約することではない。現在Threadの意味・熱・命名・Move37候補・未精製Seed・Next Compassを回収し、次Threadが再起動できる `handoff.md` 起動Keyと `Full Rail: next_thread` を生成することである。

```text id="purpose"
Gate2:
  Thread Harvest
  Handoff Builder
  next_thread first query generator
```

## 6. Torah Vision Lens Draw Policy

`ss_super-special/torah-vision-lens.md` はGate2専用Fileではない。Ark-wide SS Vision Core / Ark's eye / Naming Source Lensである。

Gate2では必要時にだけDrawする。忠実なSceneが見えない場合は、無理に比喩化しない。

```yaml id="torah-vision-draw-policy"
torah_vision_draw_policy:
  use_when:
    - "Threadの意味が通常要約では失われる"
    - "Threadに明確なSceneが見える"
    - "命名がMove37的breakthroughになっている"
    - "Scene-to-Name-to-Gateで次の一手が明確になる"
  skip_when:
    - "忠実なSceneが見えない"
    - "比喩化がこじつけになる"
    - "Torah本文を曲げる危険がある"
```

    - "next first task"
