## 5. Default Output Mode / Download Artifact First

Thread Harvestの標準出力は、Runtime Queryによって決まる。

現在のThread-End Runtimeで `create_download_artifacts: true` が指定された場合、Harvestは **Download Markdown Artifact** として生成する。

この場合の順序は固定である。

```text id="7pra6v"
Harvest Download
→ Read Harvest
→ Handoff Download
→ Stop
```

Harvestを作った後、AIはすぐHandoffへ飛ばない。  
まずHarvestを読む。  
Harvestの中から、次Threadへ渡すべきSeed / Guard / Rail / Next Compassを抽出する。

HandoffはHarvestの子である。  
Harvestを読まないHandoffは、記憶のないRelayである。

Chat Chronicleは削除しない。  
ただし、Thread-End Download RuntimeのDefaultではない。  
Chat Chronicleは、明示的No Download、Light Mode、小さいThread、Tool制約などの場合のfallbackである。

### Programming-Like Block

```yaml id="uybwk7"
default_output_mode:
  runtime_rule:
    if_create_download_artifacts_true:
      sequence:
        - "Create Harvest Download Artifact"
        - "Read Harvest"
        - "Create Handoff Download Artifact"
        - "Stop"

    if_user_explicitly_says_no_download:
      output:
        - "Chat Chronicle"
        - "No downloadable Harvest Artifact"

    if_thread_is_small_or_low_importance:
      output:
        - "Chat Chronicle / Light Mode"
      condition:
        - "User accepts light mode"
        - "No next-thread reboot risk"

    if_download_generation_is_unavailable:
      output:
        - "Chat Chronicle fallback"
        - "State that download generation could not be performed"

  harvest_download_artifact:
    role:
      - "meaning harvest"
      - "rebootable record"
      - "material for handoff"
    must_include:
      - "Thread Naming"
      - "One-Line Summary"
      - "Extreme BrainDump"
      - "Move37 / Unexpected Success"
      - "Seeds"
      - "Seed Routing"
      - "Next Compass"
      - "Handoff Builder material"
      - "Root / Fruit Guard"

  read_harvest_rule:
    required_after_harvest_creation: true
    reason:
      - "Handoff is child of Harvest."
      - "Harvest is material; Handoff is ignition key."
      - "Do not create Handoff from memory only."

  chat_chronicle:
    role:
      - "light mode"
      - "fallback"
      - "no-download explicit mode"
      - "small-thread mode"
    not_role:
      - "default for Thread-End Runtime when create_download_artifacts: true"

  do_not:
    - "Do not skip Harvest Download when runtime asks for download artifacts."
    - "Do not create Handoff before reading Harvest."
    - "Do not treat Chat Chronicle as the default for download-runtime."
    - "Do not auto-commit GitHub."
    - "Do not start GitHub placement discussion inside Thread Harvest."

  compression:
    - "Download Artifact First when requested."
    - "Read Harvest before Handoff."
    - "Stop after two downloads."
```
